


## epayBot-易支付机器人拓展插件

为你的彩虹易支付添加telegram机器人通知功能

- 本代码为BiHuo钱包三方插件，对接彩虹易支付使用
- 代码自动将易支付所有支付/结算订单推送至某个地址，大家也可用于其他推送
- 
- 无需修改易支付任何代码，外部无权限查询站点数据
- 支持结算通知、订单通知
- 开源代码，无冷门包，避免后门代码



## 使用方法

- 编辑.env文件
- 如果是推送至BiHuo机器人，请在BiHuo申请Key和PushUrl并填入.env文件
- 编译：go build ~~~
- 上传文件至服务器
- 运行： ./~~~
- 后台运行使用守护进程或直接nohup
- （其实直接上传文件到服务器 nohup go run main.go & 也可以，一点小代码没啥要求）
